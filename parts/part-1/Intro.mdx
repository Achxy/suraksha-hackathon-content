# SuRaksha - Real-time Anomaly Detection

Detect tampering in land records, legal documents, and financial statements in real-time.

## What You'll Build

- A document analysis pipeline that detects forged fields
- Real-time comparison of submitted vs. original documents
- Intelligent risk scoring for underwriting decisions
- Interactive investigation dashboard

## Prerequisites

- Basic knowledge of React and TypeScript
- Familiarity with Elastic UI components
- Understanding of document verification workflows

## Exercise Flow

1. **Document Ingestion** - Load and parse documents
2. **Field Extraction** - Extract key fields using OCR simulation
3. **Tamper Detection** - Compare extracted vs. expected values
4. **Risk Assessment** - Score documents based on tamper evidence
5. **Reporting** - Generate investigation reports
